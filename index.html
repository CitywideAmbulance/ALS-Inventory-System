Create Inventory Item

Instructions:
• Select a Category from the dropdown.
• Select an Item from the dropdown.
• Press the camera icon to scan or type the Barcode (if available).
• Type the Lot number (if available).
• Choose an Expiration date (if available).
• Enter the Quantity you are adding to stock.
• Once you are finished, press Save item and the data will be transfered to the database.

Equipment Inventory

Instructions:
• Press the camera icon to scan a barcode or type the lot number/item name and press the search icon in the first column.
• Existing equipment: fields lock and the fields for Assigned Location and Current Status remains editable.
• Missing item: fill fields.
• Select a Location from the dropdown.
• Select a Status from the dropdown.
    - Active status will have new fields appear for date returned, in-service date, and upcoming service date.
    - OOS status will have new fields appear for the service reason, comments, and date shipped.
• Once you are finished, press Save Equipment and the data will be transfered to the database.

ALS Bag Restock

Scan items, enter quantities, submit

Instructions:
• First, select a bag number from the dropdown above the table.
• Press the camera icon to scan a barcode or type the lot number/item name and press the search icon in the first column.
• Existing item: fields lock and Bag Qty remains editable.
• Missing item: fill fields, choose replacement, then click Add & Use.
• Once you are finished, press Submit All and the data will be transfered to the database.

Narcotics

Instructions:
• First, select a Mode from the dropdown.
